⚔️ 致命一击

Improved Critical

通用专长 【战斗】

专长概要

类型 通用专长
先决条件 专长武器专攻 (Weapon Focus)、战斗技巧 (Combat Expertise)、BAB +4
益处 重击威胁范围翻倍
特殊 可多次选择，每次作用于不同武器
来源 PHB p. 95
适用 战士、特定构建

专长描述

你能够更精准地找到敌人的弱点，使你的攻击更容易造成致命伤害。

📋 先决条件

  • 专长武器专攻 (Weapon Focus) - 必须已选择同一种武器
  • 战斗技巧 (Combat Expertise)
  • 基本攻击加值 (BAB) +4

说明：选择一种你已获得武器专攻加值的武器。你必须以同一种武器选择武器专攻和致命一击。

✅ 益处

使用选定武器攻击时，重击威胁范围翻倍。

  • 原本重击范围 20 → 威胁范围 19-20
  • 原本重击范围 19-20 → 威胁范围 17-20（需要锐锋法术或锐锋属性）
  • 原本重击范围 18-20 → 威胁范围 15-20

注意：重击倍率不增加，只有威胁范围翻倍。

机制详解

重击判定流程

  1. 攻击掷骰：掷 d20 + BAB + 属性调整值 + 其他加值
  2. 判定命中：结果 ≥ 目标 AC
  3. 判定威胁：如果掷骰结果在重击威胁范围内 → 重击威胁
  4. 确认重击：进行一次额外攻击掷骰（只掷 d20 + 加值，不加其他）
  5. 判定确认：如果确认掷骰结果 ≥ 目标 AC → 重击成功
  6. 计算伤害：基础伤害 × 重击倍率

重击威胁范围示例

武器 原本威胁范围 获得致命一击后 重击倍率
长剑 (Longsword) 19-20 17-20 ×2
巨剑 (Greatsword) 19-20 17-20 ×2
轻弩 (Light Crossbow) 19-20 17-20 ×2
战斧 (Battleaxe) 20 19-20 ×3
重锤 (Heavy Flail) 20 19-20 ×2

特殊情况

  • 重击范围已经是 18-20 的武器：获得致命一击后变为 15-20（威胁概率 30%）
  • 重击范围已经是 17-20 的武器：获得致命一击后变为 13-20（威胁概率 40%）
  • 无法再翻倍：如果威胁范围已经达到 13-20，则无法再通过致命一击翻倍（但可以通过锐锋法术或锐锋属性扩展到 11-20）

📖 实战示例

场景：8级人类战士（BAB +8），使用长剑，已选择武器专攻（长剑）和致命一击（长剑）。

攻击：

  1. 第一次攻击：掷 d20 → 17 + 8（BAB） + 4（力量调整值） + 3（武器专攻+力量调整值） + 1（其他） = 29 命中AC 22 的敌人
  2. 重击威胁：掷骰 17 在威胁范围 17-20 内 → 重击威胁！
  3. 确认重击：掷 d20 → 12 + 8（BAB） + 4（力量调整值） + 3（武器专攻+力量调整值） + 1（其他） = 28 ≥ 22 → 重击成功！
  4. 计算伤害：基础伤害 1d8 + 4（力量调整值） + 3（武器专攻+力量调整值） + 1（其他） = 1d8 + 8，重击时 ×2 = 2d8 + 16

分析：致命一击让重击威胁概率从 10%（19-20）提高到 20%（17-20），显著增加了重击频率。

搭配专长与构建

🔗 推荐搭配专长

  • 武器专攻 (Weapon Focus) - 先决条件，提高命中
  • 猛力攻击 (Power Attack) - 提高伤害，配合重击
  • 顺劈斩 (Cleave) - 重击后触发顺劈斩
  • 致命一击 (Improved Critical) - 本专长
  • 高等武器专攻 (Greater Weapon Focus) - 进一步提高命中
  • 武器专精 (Weapon Specialization) - 提高伤害

⚔️ 推荐构建

1. 重击战士 (Critical Warrior)

  • 专攻：武器专攻 → 战斗技巧 → 致命一击 → 猛力攻击 → 顺劈斩
  • 特点：重击频率高，适合使用重击范围大的武器
  • 推荐武器：长剑（19-20/×2）、巨剑（19-20/×2）

2. 重武器专家 (Heavy Weapon Specialist)

  • 专攻：武器专攻 → 武器专精 → 致命一击 → 猛力攻击 → 顺劈斩
  • 特点：使用重击倍率高的武器（×3、×4）
  • 推荐武器：战斧（20/×3）、重锤（20/×2）

规则冲突标注

⚠️ 规则冲突：重击威胁范围的上限

冲突点：致命一击能否将重击威胁范围翻倍到超过 11-20？

规则出处：

  • PHB p. 95：致命一击将重击威胁范围翻倍（原文："double the threat range"）
  • DMG p. 28：重击威胁范围最小为 11-20（即威胁概率最大为 50%）

冲突描述：如果武器原本重击范围是 13-20（威胁概率 40%），获得致命一击后理论上应变为 7-20（威胁概率 70%），但 DMG 规定上限为 11-20（50%）。

常见解释：

  1. 严格解释：致命一击可以翻倍，但威胁范围不能超过 11-20（DMG p. 28）
  2. 宽松解释：致命一击可以翻倍到 7-20 或更低，DMG 的上限只适用于锐锋法术和锐锋属性
  3. 折中解释：致命一击和锐锋法术/属性可以叠加，但总威胁范围不能超过 11-20

建议：与 DM 确认使用哪种解释。建议在游戏中采用严格解释（威胁范围不超过 11-20），以保持平衡。

⚠️ 规则冲突：致命一击与锐锋法术/属性的叠加

冲突点：致命一击和锐锋法术/属性能否叠加？

规则出处：

  • PHB p. 95：致命一击将重击威胁范围翻倍
  • PHB p. 246：锐锋法术将重击威胁范围翻倍
  • DMG p. 28：重击威胁范围最小为 11-20

冲突描述：如果武器同时有致命一击和锐锋法术/属性，威胁范围如何计算？

常见解释：

  1. 叠加解释：致命一击和锐锋可以叠加，威胁范围翻倍后再翻倍（例如 19-20 → 17-20 → 13-20）
  2. 不叠加解释：致命一击和锐锋效果相同，只应用一次
  3. DMG解释：叠加但不超过 11-20 上限

建议：与 DM 确认。建议采用DMG解释（叠加但不超过 11-20）。

规则出处

  • 玩家手册 (PHB) p. 95 - 致命一击专长描述
  • 地下城主指南 (DMG) p. 28 - 重击威胁范围上限
  • Complete Warrior p. 103 - 致命一击的变体规则
