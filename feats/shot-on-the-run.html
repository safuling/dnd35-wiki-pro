🏹 移动射击

专长概要

专长概要
名称移动射击 (Shot on the Run)
类型战士专长
先决条件敏捷 13、闪避、灵活移动、近程射击、基本攻击加值 +4
来源玩家手册（PHB）

效果

  • 使用远程武器攻击动作时，可在移动中间进行一次远程攻击。
  • 总移动距离不能超过你的速度。

覆盖说明

本标准摘要用于统一 PHB 专长详情页模板；下方保留本站原有的机制详解、战术说明或规则冲突记录。

Shot on the Run

通用专长 【射击】

专长概要

类型 通用专长
先决条件 精准射击 (Precise Shot)、BAB +4
益处 移动中射击不触发借机攻击
特殊 只能选择一次
来源 PHB p. 101
适用 游侠、特定构建

专长描述

你能够在移动中进行射击，而不会触发借机攻击。

📋 先决条件

  • 精准射击 (Precise Shot)
  • 基本攻击加值 (BAB) +4

说明：移动射击是精准射击的进阶专长，需要更高的BAB。

✅ 益处

在单次攻击动作中，你可以：

  1. 移动：移动最多你的速度距离
  2. 射击：进行一次远程攻击
  3. 撤离：继续移动剩余的移动距离

注意：你在移动和射击时不会触发借机攻击。

机制详解

移动射击流程

  1. 声明使用移动射击：在你的移动动作中声明
  2. 移动：移动最多你的速度距离
  3. 射击：进行一次远程攻击（使用你的攻击加值）
  4. 撤离：继续移动剩余的移动距离
  5. 结束移动：你的移动动作结束

重要规则

  • 不触发借机攻击：移动射击允许你在移动和射击时不会触发借机攻击
  • 仅限于远程攻击：移动射击只能用于远程攻击，不能用于近战攻击或法术
  • 消耗移动动作：移动射击消耗你的移动动作，你不能在同一回合内使用其他移动动作

📖 实战示例

场景：6级游侠（BAB +6），速度 30英尺，已选择移动射击。

攻击：

  1. 移动：移动 20英尺 调整位置
  2. 射击：进行一次远程攻击（使用长弓）
  3. 撤离：继续移动 10英尺 撤离到安全位置
  4. 结果：射击敌人后安全撤离，不触发借机攻击

分析：移动射击非常适合游侠等需要机动性的远程职业，可以在射击后安全撤离。

搭配专长与构建

🔗 推荐搭配专长

  • 精准射击 (Precise Shot) - 先决条件，射击邻近敌人的目标无惩罚
  • 移动射击 (Shot on the Run) - 本专长，移动中射击
  • 快速射击 (Rapid Shot) - 单次攻击动作发射两支箭矢
  • 多重射击 (Manyshot) - 单次攻击动作发射两支箭矢（同一目标）
  • 近程射击 (Point Blank Shot) - 近距离射击加值

🏹 推荐构建

1. 机动射手 (Mobile Archer)

  • 专攻：近程射击 → 精准射击 → 移动射击 → 快速射击
  • 特点：高机动性，适合游击战
  • 推荐武器：长弓（1d8伤害）

2. 游击射手 (Guerrilla Archer)

  • 专攻：精准射击 → 移动射击 → 快速射击 → 多重射击
  • 特点：高机动性，适合游击战
  • 推荐武器：短弓（1d6伤害，轻型）

规则冲突标注

⚠️ 规则冲突：移动射击与快速射击的兼容

冲突点：移动射击能否与快速射击同时使用？

规则出处：

  • PHB p. 101：移动射击专长描述
  • PHB p. 99：快速射击专长描述

冲突描述：移动射击和快速射击都需要在攻击动作中使用，能否在同一回合内使用？

常见解释：

  1. 兼容解释：可以同时使用，但移动射击的射击部分可以使用快速射击
  2. 不兼容解释：移动射击和快速射击是互斥的，只能选择一种在回合内使用

建议：与 DM 确认。建议采用不兼容解释（移动射击和快速射击不能同时用于同一次攻击动作）。

规则出处
