🏹 多重射击

Manyshot

通用专长 【射击】

专长概要

类型 通用专长
先决条件 快速射击 (Rapid Shot)、BAB +6
益处 单次攻击动作发射两支箭
特殊 仅限于远程弓箭武器
来源 PHB p. 97
适用 游侠、特定构建

专长描述

你能够在一次攻击动作中发射多支箭矢，对单个目标造成多次伤害。

📋 先决条件

  • 快速射击 (Rapid Shot)
  • 基本攻击加值 (BAB) +6

说明：多重射击是快速射击的进阶专长，需要更高的BAB。

✅ 益处

在单次攻击动作中，你可以发射两支箭矢而不是一支。

  • 这两支箭矢都攻击同一个目标
  • 每支箭矢独立进行攻击掷骰和伤害掷骰
  • 你承受-4 惩罚（两支箭矢各-2，或总共-4，取决于DM解释）

注意：多重射击只适用于弓箭武器（短弓、长弓、复合弓）。

机制详解

多重射击流程

  1. 声明使用多重射击：在你的攻击动作中声明
  2. 计算攻击加值：每支箭矢的攻击加值 = BAB + 敏捷调整值 + 其他 - 4（多重射击惩罚）
  3. 进行攻击掷骰：为每支箭矢独立掷骰
  4. 计算伤害：每支箭矢独立计算伤害

重要规则

  • 仅限于弓箭：多重射击只适用于弓箭武器，不适用于弩或投掷武器
  • 单次攻击动作：多重射击消耗一次攻击动作，但可以发射两支箭矢
  • 同一目标：两支箭矢必须攻击同一个目标
  • 独立判定：每支箭矢的攻击和伤害独立判定

📖 实战示例

场景：6级游侠（BAB +6），敏捷 18（+4），使用长弓，已选择快速射击和多重射击。

攻击：

  1. 第一支箭矢：BAB +6 + 敏捷调整值 +4 - 4（多重射击惩罚） = +6 攻击加值
  2. 第二支箭矢：BAB +6 + 敏捷调整值 +4 - 4（多重射击惩罚） = +6 攻击加值
  3. 伤害：每支箭矢 1d8 + 4（敏捷调整值） = 1d8 + 4
  4. 总伤害潜力：2-20 点伤害（每支箭矢 1-12，平均 6.5 × 2 = 13）

分析：多重射击显著提高了对单一目标的伤害输出潜力，但惩罚较高，需要较高的BAB和敏捷来抵消。

搭配专长与构建

🔗 推荐搭配专长

  • 快速射击 (Rapid Shot) - 先决条件，单次攻击动作发射两支箭矢
  • 多重射击 (Manyshot) - 本专长，单次攻击动作发射两支箭矢
  • 近程射击 (Point Blank Shot) - 近距离射击加值
  • 移动射击 (Shot on the Run) - 移动中射击
  • 精准射击 (Precise Shot) - 射击邻近敌人的目标无惩罚

🏹 推荐构建

1. 多重射手 (Manyshot Archer)

  • 专攻：近程射击 → 快速射击 → 多重射击 → 移动射击
  • 特点：对单一目标造成极高伤害
  • 推荐武器：长弓（1d8伤害）、复合长弓（1d8伤害，需要力量）

2. 机动射手 (Mobile Archer)

  • 专攻：近程射击 → 精准射击 → 快速射击 → 多重射击 → 移动射击
  • 特点：机动性强，适合游击战
  • 推荐武器：短弓（1d6伤害，轻型）

规则冲突标注

⚠️ 规则冲突：多重射击的惩罚计算

冲突点：多重射击的 -4 惩罚是每支箭矢各-2，还是总共-4？

规则出处：

  • PHB p. 97：多重射击专长描述（未明确说明惩罚分配方式）
  • 一些在线资源：认为每支箭矢各-2

冲突描述：多重射击的惩罚分配方式存在争议。

常见解释：

  1. 统一惩罚解释：总共-4惩罚，两支箭矢都使用这个惩罚后的攻击加值
  2. 分摊惩罚解释：每支箭矢各-2惩罚，独立计算攻击加值

建议：与 DM 确认。建议采用统一惩罚解释（总共-4惩罚），因为这样更简单。

⚠️ 规则冲突：多重射击与快速射击的叠加

冲突点：多重射击和快速射击能否叠加？

规则出处：

  • PHB p. 97：多重射击专长描述
  • PHB p. 99：快速射击专长描述

冲突描述：多重射击和快速射击都允许在单次攻击动作中发射多支箭矢，能否叠加？

常见解释：

  1. 不叠加解释：多重射击和快速射击是互斥的，只能选择一种在回合内使用
  2. 叠加解释：多重射击和快速射击可以叠加，但惩罚会累加

建议：与 DM 确认。建议采用不叠加解释（多重射击和快速射击不能同时用于同一次攻击动作）。
