⚔️ 双武器战斗

Two-Weapon Fighting

通用专长 【战斗】

专长概要

类型 通用专长
先决条件 敏捷 15+
益处 双持武器时惩罚减少2
特殊 可多次选择（双武器战斗专长树）
来源 PHB p. 103
适用 游荡者、特定构建

专长描述

你擅长同时使用两把武器进行攻击，能够更有效地发挥双武器战斗的威力。

📋 先决条件

  • 敏捷 (Dexterity) 15+

说明：双武器战斗需要较高的敏捷来协调两把武器的攻击。

✅ 益处

当你双持武器战斗时，攻击加值惩罚减少2。

双武器战斗惩罚表

攻击类型 无专长 有专长
主手攻击 -6 -4
副手攻击（额外攻击） -10 -8
副手轻型武器（额外攻击） -6 -4

注意：如果副手武器是轻型武器，惩罚再减少2（即主手-4，副手-4）。

机制详解

双武器战斗流程

  1. 选择双武器：主手武器 + 副手武器（副手武器必须是轻型武器以获得完整加值）
  2. 计算攻击加值：主手攻击 = BAB + 力量调整值 + 其他 - 双武器惩罚；副手攻击 = BAB + 力量调整值 + 其他 - 双武器惩罚
  3. 进行攻击：先攻击主手，然后攻击副手（额外攻击）
  4. 计算伤害：主手武器伤害 + 力量调整值；副手武器伤害 + 力量调整值的一半（向下取整）

重要规则

  • 副手伤害：使用副手武器攻击时，伤害只加一半力量调整值（向下取整）
  • 轻型武器：副手使用轻型武器时，双武器惩罚减少2
  • 额外攻击：双武器战斗允许你在主手攻击后进行一次额外攻击（使用副手武器）
  • 不可移动攻击：双武器战斗攻击必须在你的回合内，且不能移动（除非有特殊能力）

📖 实战示例

场景：4级游荡者（BAB +3），敏捷 16（+3），力量 12（+1），双持短剑（主手）+ 短剑（副手，轻型）。

有双武器战斗专长：

  1. 主手攻击：BAB +3 + 敏捷调整值 +3 + 其他 -4（双武器惩罚） = +2 攻击加值
  2. 副手攻击：BAB +3 + 敏捷调整值 +3 + 其他 -4（双武器惩罚，轻型武器） = +2 攻击加值
  3. 伤害：主手 1d6 + 1（力量调整值的一半，向下取整）；副手 1d6 + 0（力量调整值的一半，向下取整）

分析：双武器战斗专长显著减少了惩罚，使游荡者能够进行两次有效的攻击。

搭配专长与构建

🔗 双武器战斗专长树

  • 双武器战斗 (Two-Weapon Fighting) - 本专长，减少惩罚
  • 改进双武器战斗 (Improved Two-Weapon Fighting) - 先决条件：双武器战斗、BAB +6；益处：副手可以进行第二次额外攻击
  • 高等双武器战斗 (Greater Two-Weapon Fighting) - 先决条件：改进双武器战斗、BAB +11；益处：副手可以进行第三次额外攻击
  • 双武器闪避 (Two-Weapon Defense) - 先决条件：双武器战斗；益处：AC +1（当双持武器时）

⚔️ 推荐构建

1. 双武器游荡者 (Two-Weapon Rogue)

  • 专攻：双武器战斗 → 改进双武器战斗 → 高等双武器战斗
  • 特点：每回合多次攻击，适合偷袭
  • 推荐武器：短剑 + 短剑（轻型）

2. 双武器战士 (Two-Weapon Fighter)

  • 专攻：双武器战斗 → 双武器闪避 → 改进双武器战斗
  • 特点：攻守兼备，适合前排
  • 推荐武器：长剑 + 短剑

规则冲突标注

⚠️ 规则冲突：双武器战斗与猛力攻击/顺劈斩的兼容

冲突点：双武器战斗能否与猛力攻击或顺劈斩同时使用？

规则出处：

  • PHB p. 103：双武器战斗专长描述
  • PHB p. 98：猛力攻击专长描述
  • PHB p. 92：顺劈斩专长描述

冲突描述：双武器战斗、猛力攻击、顺劈斩都需要在攻击动作中使用，能否在同一回合内使用多个？

常见解释：

  1. 兼容解释：可以同时使用，但每个攻击独立计算（例如：双武器战斗的主手攻击可以使用猛力攻击，副手攻击不能使用）
  2. 不兼容解释：双武器战斗和猛力攻击/顺劈斩是互斥的，只能选择一种在回合内使用
  3. 折中解释：双武器战斗可以和猛力攻击同时使用，但不能和顺劈斩同时使用（因为顺劈斩需要击杀敌人后才能触发额外攻击）

建议：与 DM 确认。建议采用兼容解释（双武器战斗的主手攻击可以使用猛力攻击）。

⚠️ 规则冲突：副手伤害的力量调整值计算

冲突点：副手伤害是否加力量调整值的一半（向下取整）？

规则出处：

  • PHB p. 135：副手伤害只加一半力量调整值（向下取整）
  • 一些在线资源：认为副手伤害应加完整力量调整值

冲突描述：副手伤害的力量调整值计算方式存在争议。

常见解释：

  1. PHB解释：副手伤害只加一半力量调整值（向下取整）
  2. 错误解释：副手伤害加完整力量调整值

建议：遵循 PHB 规则（副手伤害只加一半力量调整值）。
