🎲 基础概念

骰子系统

D&D 3.5e 使用多种多面骰：

  • d4：4面骰，常用于短剑伤害
  • d6：6面骰，常用于短弓伤害、熊猫等生命骰
  • d8：8面骰，常用于长剑伤害
  • d10：10面骰，常用于战戟伤害
  • d12：12面骰，野蛮人生命骰
  • d20：20面骰，核心检定骰
  • d100（d%）：百分骰，由两个d10组成

检定（Check）

标准检定公式：

d20 + 调整值 ≥ 难度等级（DC）= 成功

常见检定类型：

  • 技能检定：d20 + 技能等级 + 属性调整值
  • 攻击掷骰：d20 + 基础攻击加值 + 属性调整值 + 其他加值
  • 豁免检定：d20 + 豁免加值 + 属性调整值
  • 属性检定：d20 + 属性调整值

难度等级（DC）参考

任务难度DC值示例
极易0走路不摔倒
容易5攀爬绳子（有绳结）

⚠️ 自然1与自然20：d20检定中，自然1（未加调整值为1）在攻击掷骰中永远失误，在豁免检定中永远失败；自然20（未加调整值为20）在攻击掷骰中永远命中（且可能重击）。注意：技能检定不适用自然1/20规则。

💪 六大属性值

每个角色有六项核心属性，取值通常在 3-18 之间（标准点购法上限18）。调整值 = (属性值 - 10) ÷ 2，向下取整。

力量
Strength (STR)

近战攻击/伤害加值，负重能力，力量检定（踹门/破门）

敏捷
Dexterity (DEX)

AC加值，远程攻击加值，反射豁免，灵巧类技能

体质
Constitution (CON)

每级生命值加值，强韧豁免，维持专注（被击中时）

智力
Intelligence (INT)

法师施法属性，每级技能点加值，知识类技能，语言数量

感知
Wisdom (WIS)

牧师/德鲁伊施法属性，意志豁免，感知类技能

魅力
Charisma (CHA)

术士/圣武士施法属性，社交类技能，驱散不死

⚔️ 战斗规则

战斗轮次

一个战斗轮（Round）约等于6秒，每位参与者按先攻顺序（Initiative）依次行动。

先攻掷骰 = d20 + 敏捷调整值 + 其他加值

动作类型

每轮，一个角色可以执行以下动作的组合：

  • 标准动作（Standard Action）：攻击、使用法术等主要行动
  • 移动动作（Move Action）：移动最多速度距离，拾取物品等
  • 整轮动作（Full-Round Action）：取代标准+移动；如全力攻击
  • 自由动作（Free Action）：说话（简短）、释放武器等，次数不限
  • 即时动作（Immediate/Swift Action）：部分特殊能力
  • 5英尺步进：不引发借机攻击的小距离移动（只能用1次/轮）

近战攻击

近战攻击掷骰 = d20 + 基础攻击加值（BAB）+ 力量调整值 + 武器加值

若 攻击掷骰 ≥ 目标AC，则命中。

命中后伤害 = 武器伤害骰 + 力量调整值（×1.5 若双手持握）+ 武器加值

AC计算公式：

AC = 10 + 护甲加值 + 盾牌加值 + 敏捷调整值 + 体型加值 + 其他加值

重击（Critical Hit）

攻击掷骰出现重击威胁范围（默认自然20）时，进行确认掷骰（再次攻击掷骰 ≥ 目标AC），确认后造成重击伤害（倍率×2或×3）。

常见武器重击属性：长剑 19-20/×2 | 战镰 18-20/×2 | 匕首 19-20/×2 | 战锤 20/×3 | 战戟 20/×3 | 斧矛 20/×4

借机攻击（Attack of Opportunity）

以下情况会引发借机攻击：

  • 在威胁范围内进行移动
  • 在威胁范围内施放法术（不含"快速法术"）
  • 在威胁范围内使用远程武器
  • 拾取地面物品
  • 站起（从倒地状态）

每轮默认只能进行1次借机攻击（专长"武器警觉"可提升）。

✨ 魔法规则

法术施放基础

  • 准备型法术（法师、牧师、德鲁伊等）：每日准备，施放后消耗
  • 自发施法（术士、吟游诗人等）：已知法术可随时施放，受每日次数限制
  • 施法属性：法师→智力，牧师/德鲁伊→感知，术士/吟游诗人→魅力

施法豁免难度（Spell DC）

法术DC = 10 + 法术环级 + 施法属性调整值 + 其他加值

目标进行对应豁免检定，若结果 ≥ DC 则成功豁免（通常减半伤害或完全免疫）。

法术成分

  • 言语成分 (V)：需要能够说话，沉默法术区域内无法施放
  • 姿势成分 (S)：需要手部自由，束缚时无法施放
  • 材料成分 (M)：需要消耗指定材料
  • 法器成分 (F)：需要持有特定物品（不消耗）
  • 神圣/法术专注 (DF)：牧师/德鲁伊的神圣专注
  • 额外法术位 (XP)：部分强力法术消耗施法者经验值

法术抵抗（SR）与法术免疫

部分生物拥有法术抵抗（Spell Resistance, SR）。施法者需通过施法者等级检定才能使法术生效：

d20 + 施法者等级 ≥ 目标SR = 法术有效

🛡️ 豁免检定

三大豁免

豁免类型主要属性对抗效果
强韧（Fortitude）体质毒素、疾病、能量吸取、即时死亡效果
反射（Reflex）敏捷区域爆炸（火球等）、陷阱、毁灭射线
意志（Will）感知精神控制、迷惑、恐惧、幻象效果
豁免掷骰 = d20 + 基础豁免加值 + 属性调整值 + 其他加值 ≥ 法术DC

🎯 技能规则

技能点分配

每升1级获得技能点数 = 职业技能点 + 智力调整值（最低1点）。

  • 本职技能：上限 = 角色等级 + 3，每点技能点提升1级
  • 跨职业技能：上限 = (角色等级 + 3) ÷ 2，每2点技能点提升1级

协助技能（Aid Another）

一名角色可以帮助另一名角色进行技能检定：辅助者先进行该技能的DC10检定，成功后，被辅助者的技能检定获得+2加值。
