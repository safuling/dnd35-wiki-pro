角色创建指南

Character Creation Guide

创建角色是 D&D 游戏的第一步。本指南将带你完成从选择种族到确定最终属性的完整流程。

标准创建流程

  1. 选择种族：从 种族列表 中选择一个种族。种族决定了你的体型、速度、种族特性和属性加成。
  2. 选择职业：从 职业列表 中选择一个职业。职业决定了你的 HD、BAB、豁免、技能点和职业能力。
  3. 确定属性值：使用掷骰、购点或标准阵列法确定六项属性（力量、敏捷、体质、智力、感知、魅力）。
  4. 选择技能：根据职业和智力调整值分配技能点数。参见 技能列表。
  5. 选择专长：1级时获得1个专长（人类额外+1）。参见 专长列表。
  6. 购买装备：根据起始金币购买武器、盔甲、冒险装备等。参见 装备列表。
  7. 确定细节：计算 HP、AC、攻击加值、豁免检定、先攻调整值，并编写角色背景。

新手推荐组合

如果你是新手玩家，以下是六个经典的种族+职业组合，容易上手且在游戏中表现出色：

人类战士

高 HP、多种武器和盔甲选择、额外专长。适合喜欢近战和简单直接战斗风格的玩家。

精灵法师

高智力加成、法术多样性。适合喜欢策略和远程魔法攻击的玩家，但需要注意脆弱的 HP。

矮人牧师

高体质和感知、治疗能力、重甲熟练。适合喜欢支援队友和持久战的玩家。

半身人游荡者

高敏捷加成、偷袭伤害、丰富技能。适合喜欢潜行、侦查和精准打击的玩家。

半精灵吟游诗人

多才多艺、吟游音乐、部分法术。适合喜欢社交、辅助和灵活应变的玩家。

侏儒术士

魅力加成、自发施法、幻术专精。适合喜欢即兴发挥和创意法术使用的玩家。

常见创建错误

应避免的常见错误

属性分配不当：把高属性放在不重要的属性上。例如，战士应该优先力量和体质，而不是魅力。

忽视种族限制：某些职业有阵营限制（如圣武士必须守序善良）。

技能点计算错误：1级时技能点 = (职业基础 + INT 调整值) × 4，之后每级 = (职业基础 + INT 调整值) × 1。

忘记种族特性：精灵的武器熟练、矮人的抗毒性、半身人的幸运等经常被遗忘。

装备超支：确保起始金币没有超过职业规定范围。

忽视负重：购买太多装备可能导致速度降低和检定惩罚。
