战斗规则 Combat

战斗是D&D冒险的核心组成部分。D&D 3.5e采用回合制战斗系统，以6秒为一个轮（Round）。每个角色在自己的回合中可以执行特定类型的动作。本页详细涵盖战斗流程、动作类型、攻击规则、特殊攻击、状态条件以及相关特殊能力的完整规则。

核心参考：《玩家手册》第八章 "Combat"、《城主指南》相关章节。

一、战斗流程

战斗开始：确定突袭

战斗开始时，首先判断是否存在突袭轮（Surprise Round）。

  • 如果一方注意到另一方而另一方没有注意到，则产生突袭轮。
  • 未被突袭的角色进行正常的先攻检定。
  • 在突袭轮中，未被突袭的角色可以执行一个标准动作或一个移动动作（不能两者都做），以及自由动作。
  • 被突袭的角色在突袭轮中不能行动，且处于措手不及（Flat-footed）状态——失去AC上的敏捷加值。
  • 用侦查 vs 隐匿对抗来确定是否发现对方。
措手不及 (Flat-footed)：战斗开始时，所有角色都处于措手不及状态，直到他们第一次行动。措手不及的角色失去AC上的敏捷加值（如果有的话），但保留其他AC加值。

先攻检定

先攻检定 = 1d20 + 敏捷调整值 + 其他修正（如精通先攻专长 +4）
  • 所有参与战斗的角色投掷先攻，按从高到低顺序行动。
  • 先攻相同则按敏捷调整值从高到低排列；如果仍相同则重投。
  • 先攻顺序在整个战斗过程中保持不变。
  • 后到的角色按照其先攻结果插入序列。
延迟行动：角色可以选择延迟行动（不投先攻或自愿降低先攻位序）。延迟后角色在后续轮中使用新的先攻位序。如果多个角色都延迟，按原始先攻顺序恢复。

回合结构

┌─────────────────────────────────────────────────┐ │ 一个完整的轮 (Round = 6秒) │ ├─────────────────────────────────────────────────┤ │ │ │ 角色A的回合 → 角色B的回合 → 角色C的回合 → ... │ │ │ │ 每个角色的回合可包含： │ │ ┌──────────┐ ┌──────────┐ ┌──────────┐ │ │ │ 标准动作 │ + │ 移动动作 │ + │ 自由动作 │ │ │ └──────────┘ └──────────┘ └──────────┘ │ │ 或 │ │ ┌──────────┐ ┌──────────┐ │ │ │ 整轮动作 │ + │ 自由动作 │ │ │ └──────────┘ └──────────┘ │ │ │ │ * 任意数量自由动作 + 最多1个即时/迅捷动作 │ │ * 借机攻击可在他人回合触发 │ └─────────────────────────────────────────────────┘

每轮开始时，各种持续时间效果按顺序处理（先攻从高到低）。每轮结束时，效果持续时间减少1轮。

二、战斗动作类型

标准动作 Standard Action

标准动作是最常见的动作类型，允许角色执行一次有意义的行动。一个回合中只能执行一个标准动作（除非用整轮动作替代）。

动作 类型 借机攻击 说明
近战攻击 标准 否 使用近战武器进行一次攻击。可移动后攻击或攻击后移动。
远程攻击 标准 是 使用远程武器进行一次攻击。在敌人威胁范围内射击会引发借机攻击。
施放法术 标准 是* 施放施法时间为1个标准动作的法术。在威胁范围内施法引发借机攻击。施法时受到攻击需进行专注检定。
使用类法术能力 (Sp) 标准 是 如同施法，可被干扰，需专注检定。
使用超自然能力 (Su) 标准 否 不会被借机攻击打断，不需要专注检定。
使用特异能力 (Ex) 标准 否 不会被借机攻击打断，不需要专注检定。除非另有说明。
全防御 标准 否 AC获得+4闪避加值，持续1轮。不能与战斗防御配合使用。
战斗防御 标准 否 攻击检定-4，AC获得+2闪避加值（有战斗专长时+3），持续1轮。
使用物品 标准 视情况 激活魔法物品、使用药水/卷轴等。某些物品的使用会引发借机攻击。
援助他人 标准 否 进行一次攻击检定（AC 10），成功则使盟友的攻击或防御获得+2加值。
使用特殊能力 标准 视情况 如威吓检定、解除陷阱、使用技能等。具体视能力描述。
*施法引发借机攻击：在敌人的威胁范围内施法会引发借机攻击。攻击者可以打断施法——如果施法者受到伤害，需要进行DC=10+伤害值+法术等级的专注检定，失败则法术丢失。

移动动作 Move Action

移动动作允许角色在战场移动或执行简单的物理操作。一个回合中可以执行一个移动动作（在标准动作之外）。如果只使用移动动作而不使用标准动作，可以再执行第二个移动动作。

动作 借机攻击 说明
移动 视情况 移动至多等于速度的距离。离开敌人威胁格时引发借机攻击（进入威胁格不会）。可以5尺快步替代。
5尺快步 否 移动5英尺且不引发借机攻击。每轮只能执行一次，且不能在本轮中使用其他移动方式。
拔武器 否 拔出一件武器。BAB+1的角色可以将其作为移动动作的一部分（与移动合并）。
收武器 是 收起武器需要移动动作且引发借机攻击。
站起 是 从倒地状态站起需要移动动作且引发借机攻击。
骑乘/下马 否 骑上或从坐骑上下来。骑术检定DC 20可以作为自由动作完成。
操纵物品 视情况 开门、拾取物品、移动重物等。某些操作会引发借机攻击。
控制受惊坐骑 否 骑术检定控制受惊的坐骑。
准备或松开盾牌 是 装备或卸下盾牌。

整轮动作 Full-Round Action

整轮动作占用整个回合——角色不能进行标准动作或移动动作（但可以进行5尺快步和自由动作）。

动作 借机攻击 说明
全力攻击 否 使用全部攻击次数进行攻击。可以进行一次5尺快步。多次攻击按BAB决定（如BAB +6/+1）。
奔跑 否 移动速度×4（穿重甲时×3）。奔跑时失去AC上的敏捷加值。可以持续奔跑的轮数=体质值，超过后每轮需体质检定（DC 10，每轮+1）。
撤退 否 移动速度×2的距离，离开第一格不引发借机攻击。不能在看不见的情况下撤退。
施放整轮法术 是 施放施法时间为1整轮的法术。法术在下一轮开始时生效。
全力攻击详解：当BAB达到+6时获得第二次攻击（-5减值），+11时第三次攻击（-10减值），+16时第四次攻击（-15减值）。双武器战斗、加速术、 haste等可以额外增加攻击次数。

自由动作 Free Action

自由动作几乎不花时间，可以在执行其他动作的同时进行。每轮可以执行的自由动作数量由DM裁定（通常不超过几个）。

动作 说明
说话说几句话或下达简短命令。可以在任何时间进行（甚至不是自己的回合）。
停止专注停止维持一个需要专注的法术。
丢弃物品将手中的物品扔到地上。
倒地趴倒在地（卧倒）。
准备武器（BAB+1）BAB至少+1的角色可以将拔武器与移动合为一个移动动作。
使用某些专长/能力某些专长或职业特性标注为自由动作使用。

即时动作 / 迅捷动作 Immediate/Swift Action

即时动作和迅捷动作（3.5e后期规则补充）占用极短时间但消耗更多精力。

动作 类型 说明
延迟 即时 放弃当前回合的行动，等待更好时机。延迟后先攻位序变为实际行动的时机。
准备动作 标准 声明准备的动作和触发条件。当条件满足时（在下一轮之前），可以以即时动作执行准备好的动作。准备后先攻位序改变。
借机攻击 即时 当敌人在威胁范围内执行引发借机攻击的动作时，可以进行一次近战攻击。
借机攻击详解：
  • 每个角色每轮只能进行一次借机攻击（有战斗反射专长时可以使用DEX加值次数的借机攻击）。
  • 借机攻击中断触发动作——攻击在触发动作完成前进行。
  • 威胁范围：角色可以用近战武器攻击到的所有格子（通常邻接格，长武器更远）。
  • 引发借机攻击的常见行为：离开威胁格、在威胁范围内施法、在威胁范围内使用远程武器。
  • 不会引发借机攻击的行为：5尺快步、撤退的第一格、进入威胁格。

三、攻击规则

攻击检定

近战攻击检定 = 1d20 + BAB + STR调整值 + 体型调整值 + 其他修正
远程攻击检定 = 1d20 + BAB + DEX调整值 + 体型调整值 + 射程减值 + 其他修正

如果攻击检定结果 ≥ 目标的AC（护甲等级），则命中。

自动命中与自动失手：自然骰20（d20实际掷出20）总是命中，自然骰1总是失手。自然20同时是重击威胁。

护甲等级 (AC)

AC = 10 + 护甲加值 + 盾牌加值 + DEX调整值 + 体型调整值
    + 天生护甲加值 + 偏斜加值 + 闪避加值 + 其他修正
AC类型 英文 计算方式 说明
普通AC Armor Class 10 + 所有加值 正常情况下的防御值。
接触AC Touch AC 10 + DEX + 体型 + 闪避 + 偏斜 不计护甲、盾牌和天生护甲。用于接触攻击和某些法术。
措手不及AC Flat-footed AC 10 + 护甲 + 盾牌 + 天生 + 体型 + 偏斜 不计DEX加值和闪避加值。战斗开始或意外情况时。

伤害计算

近战伤害 = 武器伤害骰 + STR调整值（双手武器×1.5，副手×0.5）+ 其他修正
远程伤害 = 武器伤害骰 + 其他修正（复合弓可加STR加值）

伤害最低为1（即使所有减值导致计算结果为0或负数）。

重击 (Critical Hit)

当攻击掷出自然20（或其他武器特定的重击威胁范围）时，需要进行重击确认检定：

  1. 进行一次新的攻击检定（使用相同的所有加值），目标为敌人的AC。
  2. 如果确认检定命中，则造成重击伤害。
  3. 如果确认检定失手，则只造成普通伤害。

重击时，伤害骰和所有固定加值（STR加值、武器附魔加值、专长加值等）都乘以重击倍率。额外骰（如偷袭骰、火焰附魔骰）不乘以重击倍率。

武器类型 重击威胁范围 重击倍率
大多数武器20×2
长剑、细剑等19-20×2
镰刀20×4
弯刀18-20×2
战斧20×3
巨斧20×3
精通重击专长 (Improved Critical)：将武器的重击威胁范围翻倍（如19-20变为17-20）。不能与其他扩大重击范围的效果叠加。

四、特殊攻击

冲撞 (Bull Rush)

标准动作 或 整轮动作（冲锋时）

试图将目标推回而不对目标造成伤害。

  1. 步骤一：攻击者进入目标所在格（引发借机攻击，从目标以外的所有威胁者处）。
  2. 步骤二：进行力量对抗检定。攻击者获得+4加值（每比中型大一级+4），防御者每有额外腿或特别稳定+4。
  3. 步骤三：攻击者胜出，目标被推回5英尺。攻击者可以选择跟随移动（每超出4点再多推5英尺）。不能超过速度限制。
  4. 防御者胜出，攻击者被推回5英尺（如果攻击者无法后退则倒地）。

擒抱 (Grapple)

标准动作

试图抓住并控制目标。

擒抱检定 = 1d20 + BAB + STR调整值 + 特殊体型调整值
体型调整值：超巨型+16，巨型+12，超大型+8，大型+4，小型-4，超小型-8，微型-12，超微型-16
  1. 步骤一 - 抓住：进行一次近战接触攻击。失手则擒抱尝试失败（引发借机攻击）。
  2. 步骤二 - 擒抱：进行对抗擒抱检定。成功则双方进入擒抱状态。如果体型差距超过2级，大的一方自动赢得对抗。
  3. 步骤三 - 维持擒抱：维持擒抱需要每轮进行一次对抗检定。在擒抱中可以进行：伤害对手、压制、移动、使用轻武器等。
  4. 逃脱擒抱：进行擒抱检定对抗或脱逃术检定（对抗擒抱检定DC）。
擒抱状态效果：擒抱中的角色失去AC上的DEX加值（对非擒抱者），不能使用需要双手的动作，不能进行借机攻击，移动速度为0。

绊摔 (Trip)

标准动作（接触攻击）

  1. 进行一次近战接触攻击（不造成伤害）。有些武器可以代替接触攻击进行绊摔且不引发借机攻击。
  2. 进行力量对抗检定（vs 力量或敏捷中较高者）。防御者用4条腿+4稳定加值。
  3. 攻击者胜出：目标倒地（Prone）。
  4. 防御者胜出：防御者可以立刻尝试反绊摔攻击者（不需要接触攻击）。

倒地状态效果：近战攻击-4，远程攻击被+2命中，站起需要移动动作且引发借机攻击。

卸武 (Disarm)

标准动作

  1. 进行近战接触攻击（徒手卸武会引发借机攻击，用武器卸武不会）。
  2. 进行力量对抗检定。双手持武器的防御者+4；使用轻型武器的攻击者-4。
  3. 攻击者胜出：防御者失去武器。如果攻击者徒手，则接住武器。
  4. 防御者胜出：防御者可以反卸武攻击者。

闯越 (Overrun)

标准动作（与移动合用）

试图穿过敌人占据的格子。

  1. 在移动中进入目标格子（引发借机攻击）。
  2. 目标可以选择阻挡或让开。让开则攻击者正常通过。
  3. 如果阻挡：进行力量对抗检定。攻击者胜出则可以穿过。防御者胜出则攻击者被挡回并倒地。
  4. 有"精通闯越"专长的角色不会引发借机攻击，且目标不能选择让开。

夹击 (Flanking)

当两个友方角色从相对两侧（对角线也算）用近战武器威胁同一个敌人时，双方都获得+2夹击加值到近战攻击检定上。

  • 只有近战攻击可以获得夹击加值。
  • 两个威胁者必须能够从对方画一条直线穿过目标中心。
  • 偷袭能力的游荡者特别受益于夹击。
  • 拥有"精通夹击"专长时加值提升至+4（仅限对游荡者有效）。

冲锋 (Charge)

整轮动作（特殊情况可作为标准动作）

快速冲向敌人进行一次强力攻击。

  • 移动至少10英尺（不超过速度×2），直线前进。
  • 冲锋路径上不能有障碍物或困难地形。
  • 冲锋后进行一次近战攻击，攻击检定+2。
  • 冲锋后AC-2直到下一轮开始。
  • 使用长枪从坐骑上冲锋时伤害×2。

五、状态条件完整列表

以下是D&D 3.5e中所有主要状态条件的详细说明。多种状态可以同时影响一个角色，效果叠加（除非互相矛盾）。

昏迷 Unconscious 严重 HP在0或以下（濒死），或非致命伤害超过当前HP。完全无力，无助。可以被叫醒（受到伤害或泼水等），但通常自然恢复需要时间。

六、特殊能力

特异能力 (Extraordinary Abilities, Ex)

特异能力是非魔法的特殊能力，但不一定是自然或直观的。它们不会受到法术抗力影响，也不会在反魔法区域内失效。

  • 使用特异能力不会引发借机攻击（除非另有说明）。
  • 不会被干扰或打断（除非受到攻击并需要进行专注检定）。
  • 不受法术抗力影响。
  • 在反魔法区域内正常运作。
  • 不能被解除魔法。
  • 示例：野蛮人的伤害减免、龙的 frightful presence、许多怪物的特异感官（黑暗视觉、震颤感知等）。

类法术能力 (Spell-Like Abilities, Sp)

类法术能力如同施放法术一样运作，但不是真正的法术。它们有施法者等级，可以被法术抗力影响。

  • 使用类法术能力通常需要标准动作。
  • 使用类法术能力会引发借机攻击（除非另有说明）。
  • 可以被干扰（施法者受到伤害时需专注检定）。
  • 受法术抗力影响——施法者等级检定克服SR。
  • 在反魔法区域内不运作。
  • 可以被解除魔法。
  • 不能用于反制法术或被法术反制。
  • 施法者等级通常等于HD或角色等级。
  • 豁免DC = 10 + 法术等级 + CHA调整值。
  • 示例：魔鬼的暗示术、传送术，某些种族的种族类法术能力。

超自然能力 (Supernatural Abilities, Su)

超自然能力是魔法性的能力，但不像法术那样运作。它们是魔法的本质体现。

  • 使用超自然能力通常需要标准动作（除非另有说明）。
  • 使用超自然能力不会引发借机攻击。
  • 不会被干扰或打断。
  • 不受法术抗力影响。
  • 在反魔法区域内不运作。
  • 不能被解除魔法。
  • 不能被反制。
  • 施法者等级通常等于HD。
  • 豁免DC = 10 + 1/2 HD + 相关属性调整值。
  • 示例：德鲁伊的野性变形、圣武士的圣疗、龙的喷吐武器、不死生物的驱散抗性。

七、伤害减免 (Damage Reduction, DR)

基本概念

DR = 减免数值 / 克服类型
例：DR 10/魔法 表示每次受到物理伤害时减少10点，除非攻击武器是魔法武器

伤害减免在每次攻击命中后应用，从该次攻击的伤害中减去指定数值。如果伤害被减至0或以下，则该次攻击不造成任何伤害（但附加效果如毒素可能仍然生效）。

注意：伤害减免不应用于能量伤害（火焰、寒冷、电击等）——这些由抗力/免疫处理。DR只应用于物理伤害。

DR X/寒铁使用寒铁武器某些精类生物和恶魔的弱点。

八、法术抗力 (Spell Resistance, SR)

基本概念

施法者等级检定 = 1d20 + 施法者等级
如果检定结果 ≥ 目标的SR，法术正常生效
如果检定结果 < 目标的SR，法术对该目标无效

法术抗力类似于对法术的"AC"。每当法术影响到拥有SR的生物时，施法者必须进行一次施法者等级检定来克服SR。

法术抗力规则

  • 主动降低SR：拥有SR的生物可以自愿降低SR（标准动作），在该轮结束时恢复。
  • 仅影响法术和类法术能力：SR不影响超自然能力和特异能力。
  • 一次检定覆盖整个法术：如果一个法术同时影响多个有SR的生物，对每个生物分别进行SR检定。
  • 持续性法术：当有SR的生物进入持续性法术区域时，需要进行SR检定。
  • 法术描述标注：并非所有法术都受SR影响。法术描述中会标注"SR：是"或"SR：否"。
  • 穿透法术："穿透法术抗力"专长可以在SR检定上获得+2加值。
法术类型 是否受SR影响
大多数攻击性法术（火球术、闪电束等）是
大多数控制法术（支配人类、魅惑怪物等）是
力场法术（魔法飞弹、灵甲术）是
不影响生物本身的法术（解除魔法、石墙术）否
召唤/创造法术否（召唤物不受SR影响）
能量耗尽法术（如某些死灵术）通常否
不直接作用于有SR生物的区域效果否
常见SR值参考：低等级怪物SR约10-15；中等级怪物SR约15-25；高等级怪物和强大的异界生物SR可达25-35+。一般规则：SR ≈ CR + 10（粗略估计）。

九、免疫与抗性

能量抗性 (Energy Resistance)

能量抗性允许生物忽略一定量的特定类型能量伤害。与伤害减免不同，抗性不区分"类型"——只要伤害属于指定能量类型就减免。

实际伤害 = max(0, 原始伤害 - 抗性值)
抗性类型 适用伤害 常见来源
火焰抗力火焰伤害火元素、某些龙、火焰防护法术
寒冷抗力寒冷伤害冰元素、白龙、寒冷防护法术
电击抗力电击伤害风元素、蓝龙、电击防护法术
强酸抗力强酸伤害泥怪、黑龙、酸液防护法术
音波抗力音波伤害某些异界生物、音波防护法术

免疫 (Immunity)

免疫意味着生物完全不受特定类型的伤害或效果影响。免疫是绝对性的——无论伤害量多大，免疫的生物都不受影响。

抗力与免疫的区别：抗力（Resistance）是数值减免——超过抗力值的部分仍然生效。免疫（Immunity）是绝对防护——无论数值多大都完全无效。例如：火焰抗力10受到25点火焰伤害时只受15点；而火焰免疫时受到任何火焰伤害都为0。

HP与伤害状态

HP状态 效果
HP > 0正常行动，无惩罚。
HP = 0失去行动能力（Disabled）。每轮进行标准动作或移动动作后受1点伤害。只能进行自由动作。
HP = -1 到 -(CON-1)濒死（Dying）。失去意识，每轮损失1HP。可通过DC 10的医疗检定稳定（DC=10+负HP数）。稳定后每小时10%几率恢复1HP并清醒。

非致命伤害 > 当前HP昏迷（Unconscious）。非致命伤害超过当前HP时角色昏迷，直至非致命伤害被治愈或当前HP增加。
