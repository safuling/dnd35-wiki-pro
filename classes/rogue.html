盗贼（Rogue）

核心职业 PHB p.50 最多技能点

盗贼是D&D 3.5e中最灵活的技能型角色。他们擅长潜行、开锁、解除陷阱、偷窃， 并在偷袭（Sneak Attack）中造成额外伤害。在D&D 3.5e中，盗贼被认为是 非战斗解决方案的专家——许多陷阱、机关、谜题只有盗贼能处理。

虽然盗贼的HP极少（d6）且BAB极低，但他们的 每级技能点（8 + INT调整值）是所有职业中最多的，这让他们能掌握几乎所有技能。

🎯 角色定位

🔓 陷阱专家

侦察陷阱、解除陷阱、开锁。适合地城探险。

🤫 偷袭者

利用隐蔽和夹击造成额外伤害。适合暗杀构建。

🎭 社交侦察

偷听、伪装、交涉。适合城市冒险。

🗺️ 侦察兵

侦察、聆听、野外生存。适合荒野探险。

⚙️ 核心机制

偷袭（Sneak Attack）

盗贼的核心输出机制：

  • 触发条件：目标夹击（Flanked）或失去AC敏捷加值（如被夹击、无法移动、被眩晕等）。
  • 伤害骰：每1-2级 +1d6（10级+5d6，20级+10d6）。
  • 远程限制：目标必须在30英尺内，且盗贼不能隐匿失败。
  • 特殊弹药：使用"偷袭箭矢/弩箭"可以突破30英尺限制。

关键战术：与队友配合进行夹击，或利用"闷棍"等方法使敌人失去DEX加值。

技能系统（Skills）

盗贼拥有最多的技能点：每级 8 + INT调整值。

本职技能（Class Skills）：

  • 战斗相关：使用魔法装置（Use Magic Device）
  • 移动相关：攀登（Climb）、跳跃（Jump）、游泳（Swim）、潜行（Hide）
  • 侦察相关：聆听（Listen）、侦察（Spot）、搜索（Search）、察觉（Sense Motive）
  • 操作相关：开锁（Open Lock）、解除陷阱（Disable Device）、脱身（Escape Artist）
  • 社交相关：唬骗（Bluff）、外交（Diplomacy）、伪装（Disguise）
  • 知识相关：知识（地下城）（Knowledge - Dungeoneering）

策略：优先提升使用魔法装置（可以像术士一样使用法术卷轴）和解除陷阱（地城生存必备）。

🗡️ 偷袭详解

偷袭伤害成长表

等级偷袭伤害备注
1级+1d6初始能力
3级+2d6——
5级+3d6——
10级+5d6核心输出期
15级+8d6后期输出相当可观
20级+10d6终极偷袭

注意：偷袭伤害是额外骰，不受力量调整值影响。但可以通过"劲力偷袭（Power Sneak Attack）"等专长提升。

偷袭触发条件详解

以下条件满足任意一条即可触发偷袭：

  1. 目标被夹击（Flanked）：盗贼和至少一个盟友分别位于目标对立两侧。
  2. 目标失去DEX加值：
    • 被夹击（无论是否意识）
    • 无法移动（被纠缠、倒地、麻痹等）
    • 正在睡眠或失去意识
  3. 目标在盗贼的闷棍（Sap）攻击中失败（House Rule，非核心）。

无法触发偷袭的情况：

  • 目标有直觉闪避（Uncanny Dodge）能力（如游侠/武僧）。
  • 目标是构装体（Construct）、亡灵（Undead）等无生命骰的生物（除非有特殊能力）。
  • 盗贼在隐秘失败（Stealth check失败）后被发现。

📖 职业能力详解

◇ 偷袭（Sneak Attack）

详见上方"偷袭详解"板块。

◇ 直觉闪避（Uncanny Dodge）

4级起，盗贼不会因被夹击或突袭而失去AC的敏捷加值。

进阶能力（Imroved Uncanny Dodge）：8级起，免疫夹击（Flanking）。

机制意义：使盗贼在近战中更难被偷袭（其他盗贼无法对其使用偷袭）。

◇ 使用魔法装置（Use Magic Device）

盗贼的专属技能，让他们可以像术士一样使用法术卷轴和魔杖。

  • 使用法术卷轴：即使不是施法者，也能激活法术卷轴（需通过技能检定）。
  • 使用魔杖/法杖：即使不满足职业要求，也能使用魔杖（需通过技能检定）。
  • 使用魔法物品：即使不满足属性要求，也能使用魔法物品（需通过技能检定）。

构建意义：让盗贼拥有临时施法能力，弥补无法施展法术的缺陷。

💡 盗贼构建建议

潜行偷袭流（特种兵）：

  • 优先提升敏捷（DEX）和智力（INT）（技能点）。
  • 专长选择武器娴熟（Weapon Finesse） → 使用敏捷进行近战攻击。
  • 技能优先潜行、解除陷阱、使用魔法装置。
  • 战术：先手潜行 → 偷袭 → 脱离潜行 → 重复。

社交侦察流（间谍）：

  • 优先提升魅力（CHA）和智力（INT）。
  • 专长选择交涉（Negotiator） → 提升社交技能。
  • 技能优先唬骗、外交、伪装、使用魔法装置。
  • 战术：混入敌人内部 → 偷取情报 → 必要时暗杀。

🔄 多职业建议

盗贼是D&D 3.5e中最适合多职业的核心职业之一，因为技能点和偷袭机制的灵活性。

盗贼 / 战士

经典组合。战士提供高BAB和专长，盗贼提供技能和偷袭。

推荐：战士1/盗贼剩余 → 获得擅长所有护甲/武器。

盗贼 / 游侠

双专精。游侠的"双武器战斗"配合盗贼的偷袭。

需敏捷15+（双武器战斗先决）。

盗贼 / 法师（奥术诡术师进阶）

奥术诡术师（Arcane Trickster）。需要满足"能施展2环奥术"先决。

构建：盗贼3/法师3 → 奥术诡术师剩余。

盗贼 / 吟游诗人

社交间谍。吟游诗人的"鼓舞"配合盗贼的社交技能。

适合城市冒险。

🎒 推荐装备

潜行偷袭构建

  • 护甲：轻型护甲（秘银衫 +3 AC，无技能惩罚）
  • 武器：短剑（1d6，擅长）或细剑（1d6，擅长）
  • 远程：手弩（1d4，擅长）或短弓（1d6）
  • 魔法物品：潜行靴（+Hide）、无声手套（+Sneak Attack）、偷袭匕首（+Sneak Attack伤害）

社交侦察构建

  • 护甲：无（或礼服，无AC但+社交）
  • 武器：手弩（隐蔽）或细剑（优雅）
  • 魔法物品：伪装套装（+Disguise）、交涉披风（+Diplomacy）、魅力护身符（+CHA）

🏗️ 常见构建

🗡️ 潜行偷袭者

属性优先级：DEX > INT > CON

核心专长：武器娴熟、精准射击、闪避

战术：先手潜行 → 偷袭 → 脱离。

🔓 陷阱专家

属性优先级：INT > DEX > CON

核心专长：警觉（Alertness）、技能专精（Skill Focus）

战术：侦察陷阱 → 解除陷阱 → 开锁。

🎭 社交间谍

属性优先级：CHA > INT > DEX

核心专长：交涉（Negotiator）、魅惑（Persuasive）

战术：混入 → 偷取情报 → 暗杀。

⚠️ 规则冲突

关于"使用魔法装置"的先决条件

冲突描述：盗贼使用法术卷轴时，是否需要满足施法属性要求？

SRD和PHB的明确说法："You can use this skill to ... read a scroll ... even if you aren't a member of the class that normally uses that type of scroll."（SRD · Skills · Use Magic Device）

即：不需要满足施法属性要求，只需通过技能检定。

来源：SRD PHB p.84

DM解释差异：某些DM可能house rule "需要满足属性要求"。建议提前确认。

关于"偷袭"对"构装体"是否生效

冲突描述：构装体（Construct）是否有"生命骰"，是否能成为偷袭目标？

SRD的明确说法："The rogue ... can sneak attack ... creatures that do not have ... a fortitude save."（SRD · Classes · Rogue）

即：无强韧豁免的生物（如构装体、亡灵）无法被偷袭。

来源：SRD PHB p.51

扩展说明：元素（Elemental）有强韧豁免，可以被偷袭；但构装体/亡灵不可以。

关于"偷袭伤害"的计算频率

冲突描述：盗贼的偷袭伤害是"每2级+1d6"，还是"每1级+1d6"？

SRD和PHB的明确说法："+1d6 every two rogue levels beyond 1st"（SRD · Classes · Rogue）

即：每2级+1d6（1级=1d6，3级=2d6，5级=3d6， etc.）。

来源：SRD PHB p.50

常见错误：新手常误以为"每级+1d6"，导致伤害过高。请严格按SRD计算。

关于"察言观色"是否属于盗贼技能

冲突描述：SRD规定盗贼"本职技能"包含察言观色（Sense Motive），但某些第三方资料删除了该技能。

SRD的明确说法：盗贼本职技能列表包含"Sense Motive"（SRD · Classes · Rogue · Skills）。

即：察言观色是盗贼本职技能，添加时不需支付额外技能点。

来源：SRD PHB p.52

建议：社交型盗贼务必投资察言观色，配合偷袭/交涉使用。
