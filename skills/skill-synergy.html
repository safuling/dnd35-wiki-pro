一、技能共效规则说明 (Skill Synergy)

核心规则：当一个角色在某项技能上拥有 5级或更高 的技能等级（Rank）时，该角色在某些相关技能上获得 +2 协同加值（Synergy Bonus）。

这是《玩家手册》（PHB）中非常重要但经常被新手玩家忽略的规则。合理利用技能共效可以在角色构建时用最少的技能点获得最大的技能收益。

基本要点

  • 触发条件：来源技能（Source Skill）必须达到 5级（5 Ranks），不是5个技能点。技能等级 = 投入技能点 + 其他加值（如法术、物品），但共效只计算实际投入的技能等级（Ranks）。
  • 加值大小：固定为 +2。即使来源技能超过5级，加值也不会增加（除非特殊能力另有说明）。
  • 加值类型：协同加值（Synergy Bonus）。同类型的加值不叠加，但协同加值与大多数其他加值类型可以叠加。
  • 无需检定：共效加值是被动生效的，不需要进行任何检定或花费动作。
  • 跨职业技能：即使目标技能是跨职业技能（Cross-class Skill），只要来源技能达到5级，共效加值同样生效。

规则澄清：技能等级 vs 技能调整值

技能共效要求的是 5个技能等级（Ranks），不是技能调整值（Modifier）达到+5。例如，一个智力18的法师在知识(奥秘)上投入5个技能点，其技能调整为 5(等级) + 4(智力) = +9，满足共效条件。但如果只投入3个技能点，即使加上智力调整值达到+7，也不满足共效条件。

关于"知识(奥秘) + 辨识法术"双重共效

部分扩展规则提到：当角色同时拥有 知识(奥秘) 5级 和 辨识法术 5级 时，在使用魔法装置（Use Magic Device）检定上获得额外共效加值。这是特殊的"双重共效"规则，具体请参照扩展来源和DM裁定。

三、共效加值叠加规则

多个共效可以叠加吗？

答案是：视情况而定。

  • 同一来源技能的共效：每个来源技能只能提供一次+2加值，不会因为技能等级超过5级而翻倍。
  • 不同来源技能指向同一目标：如果多个不同的来源技能都为同一个目标技能提供共效，这些加值 可以叠加。因为它们来自不同的来源。

叠加示例

辨识法术 (Spellcraft) 知识(奥秘) +2, 知识(位面) +2 +4
外交 (Diplomacy) 知识(贵族与皇室) +2, 察言观色 +2 +4
使用魔法装置 (UMD) 解读文书 +2, 辨识法术 +2 +4

四、角色构建中的共效策略

高效技能构建建议

施法者推荐共效链

对于法师/术士等施法者，优先投入以下技能各5级：

  • 知识(奥秘) 5级 → 辨识法术 +2
  • 辨识法术 5级 → 使用魔法装置 +2
  • 解读文书 5级 → 使用魔法装置 +2（总计+4 UMD）
  • 知识(位面) 5级 → 辨识法术额外+2（总计+4法术辨识）

总计投入 20 个技能点，获得辨识法术+4、使用魔法装置+4的被动加值。

游荡者/斥候推荐共效链

  • 侦查 5级 → 搜索 +2
  • 搜索 5级 → 追踪 +2
  • 生存 5级 → 追踪 +2（追踪总计+4）
  • 绳技 5级 → 攀爬 +2, 脱逃 +2
  • 潜行 5级 → 躲藏 +2

交涉型角色推荐共效链

  • 知识(贵族与皇室) 5级 → 外交 +2
  • 察言观色 5级 → 外交 +2（外交总计+4）
  • 外交 5级 → 唬骗 +2
  • 唬骗 5级 → 威吓 +2, 巧手 +2, 易容 +2
  • 知识(地方) 5级 → 收集信息 +2

动物共感 (Wild Empathy)

类型职业特性（非标准技能） 关键属性CHA 魅力 可用职业德鲁伊 (Druid)、巡林客/游侠 (Ranger) 受训/未受训不适用（职业特性） 防具检定惩罚不适用

动物共感是德鲁伊和巡林客的职业特性，类似于对动物使用外交（Diplomacy）技能。可以改变动物的态度。

检定公式：1d20 + 德鲁伊/巡林客等级 + 魅力调整值

目标初始态度备注

作用距离：30英尺内。目标必须能看见和听见使用者。

总结与提醒

构建角色时的技能共效清单

  1. 先确定角色最终想使用的高优先级技能（如：使用魔法装置、外交、辨识法术）
  2. 反查共效表，找出所有能为目标技能提供+2加值的来源技能
  3. 在低等级时优先将来源技能各投入5级
  4. 注意：跨职技能的每级消耗2个技能点，需提前规划技能点预算
  5. 多个共效来源的+2加值可以叠加，这是技能系统中最强大的隐藏机制之一
